## About this repository
This repository contains the implementation used to study adaptive control for an age-structured two-species system with neural-operator-based prediction of the Lotka-Sharpe growth rate. The codebase includes:

- Operator construction and diagnostics for the structured population model.
- Simulation scripts for nominal and adaptive TAC experiments.
- Dataset generation pipelines for both the base operator-learning task and the adaptive-estimator task.
- Pretrained FNO checkpoints and the datasets used to train them.


## Installation

```bash
git clone https://github.com/lukebhan/Lotka-SharpeNeuralOperatorsforControlofPopulationPDEs.git
cd Lotka-SharpeNeuralOperatorsforControlofPopulationPDEs
python -m venv lsno_control_env
source lsno_control_env/bin/activate   # on Windows: lsno_control_env\Scripts\activate
pip install -r requirements.txt
```

Dependencies: `numpy`, `pandas`, `matplotlib`, `torch`, `neuraloperator`.

## Pretrained Resources

Pretrained datasets and models for this repository are already organized around the local `datasets/` and `models/` folders.

- Datasets: https://huggingface.co/datasets/lukebhan/NeuralOperatorsLotkaSharpePredatoryPrey
- Models: https://huggingface.co/lukebhan/NeuralOperatorsLotkaSharpePredatoryPrey

If you download or regenerate these resources separately, place dataset files in `datasets/` and model checkpoints in `models/`.

## Quick Start

After installation, you can generate the main figures directly from the provided scripts:

```bash
python scripts/make_tac_operator_figure.py # must be run before tac_adaptive or tac_sim
python scripts/make_tac_adaptive.py
```

Additional entry points:
- `python scripts/make_tac_sim_figure.py` runs the nominal TAC simulation figure.
- `python scripts/run_sim.py` runs a full simulation rollout with estimator diagnostics.
- `python scripts/gen_dataset_examples.py` visualizes representative samples from the base dataset.
- `python scripts/gen_adaptive_dataset.py` regenerates the adaptive estimator dataset.

The operator figure script reproduces the learned Lotka-Sharpe operator diagnostics, while the adaptive figure script runs the online estimator/controller loop and generates the top and bottom adaptive plots shown above.

## Training from Scratch

If you want to retrain the models, first generate the datasets, then run training.

Generate datasets:

```bash
python scripts/gen_dataset_examples.py
python scripts/gen_adaptive_dataset.py
```

The base operator-learning dataset is generated from the parametric family specification in `config.py` via `scripts/gen_dataset_examples.py`, and is saved into `datasets/simple_parametric_dataset.npz` and `datasets/simple_parametric_summary.csv`. The adaptive-estimator dataset is generated by rolling out the closed-loop adaptive simulations in parallel and is saved into `datasets/adaptive_estimator_dataset.npz` and `datasets/adaptive_estimator_summary.csv`.

Train:

```bash
python scripts/train_fno.py
python scripts/train_fno_adaptive.py
```

These scripts train the baseline and adaptive FNO predictors for the Lotka-Sharpe growth-rate operator and save checkpoints, normalization stats, and training histories under `models/`.
